What is hoomancmd?
====================

hoomancmd is a command-line interface extends python's cmd.Cmd utiling the hoomanlogic text-based human language input
framework.


Installing hoomancmd
======================

python setup.py install

Python 2.6+ or 2.7+ is required for hoomanlogic v0.1.0+


Running Unit Tests
==================

In the source tree do the following:

    python run_tests.py parsedatetime
